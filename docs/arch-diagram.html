vfs4Agent · System Architecture

系统架构

VectorStore → FUSE 虚拟文件系统 → BashRunner（优化器快速路径）→ HTTP Server / Agent REPL

BashRunner

Optimizer 快速路径
  • 命令匹配 Optimizer → 直接走 VectorStore，不起 bash 进程
  • 6 个 Optimizer：grep / find / tree / wc / ls -R / du
  • 命中失败或无匹配 → 降级到 bash spawn（fallback）
  • VFS_OPTIMIZERS 环境变量控制启用集合

FUSE Layer

虚拟文件系统桥
  • VectorStore → 真实 /vfs 挂载点
  • bash 子进程用标准 ls/cat/grep 透明访问
  • fuse-native · getattr · readdir · read
  • SIGTERM → unmount → 优雅退出

Backend

可插拔存储
  • Chroma（HTTP 服务）
  • SQLite（本地零依赖）
  • VFS_BACKEND 环境变量切换
  • VectorStoreCapabilities 声明能力
