vfs4Agent · Command Execution Flow

命令执行流程

一条命令从 Agent 发出，经 BashRunner → Optimizer 检测 → 快速路径或 bash 降级，最终返回结果的完整决策树。

Optimizer Fast Path

若 registry.find(cmd, store) 命中且 store 满足 requiredCapabilities，调用 optimizer.execute() 直接查 VectorStore，绕过 FUSE 和 bash 子进程。典型耗时 <50ms。

Bash Fallback

未命中（命令不在优化列表或 capabilities 不满足）或 optimizer 抛错时， 降级到 spawnBash(bash -c cmd)，15s 后 SIGKILL（exitCode=124）， stdout truncated to 64KB。
